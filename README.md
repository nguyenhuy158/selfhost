# selfhost

`selfhost` is a CLI tool that lists useful tools from your public GitHub repositories, presenting them in a clean, formatted table.

## Features

- **Dynamic Listing**: Fetch public repositories from GitHub.
- **Beautiful UI**: Uses `rich` for tables and progress indicators.
- **Easy to Use**: Simple CLI interface.

## Installation

```bash
pip install .
```

## Usage

```bash
selfhost
```

To see the version:

```bash
selfhost --version
```
